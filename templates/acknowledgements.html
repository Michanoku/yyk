This application was made as my submission for the final project for CS50x 2022. It was designed in VS Code using Python, HTML and CSS. As Professor Malan has said in his course many times, we should learn to stand on the shoulders of others. In this spirit, I could not have finished this application without the help and prior work of many others, all of whom shall be named here.


Database Creation

The database for this application was created by integrating radical data that I have compiled myself, with kanji data, vocabulary data and stroke order data pulled from other sources.


Kanji The kanji data was obtained from The KANJIDIC project. The KANJIDIC project files are released under a Creative Commons Attribution-ShareAlike Licence (V3.0). See the EDRDG General Dictionary Licence Statement for details. Only jouyou kanji and JLPT relevant kanji have been included.

Vocabulary The vocabulary data was obtained from jisho. Jisho uses JMdict and JMnedict for their vocabulary. These dictionary files are released under a Creative Commons Attribution-ShareAlike Licence (V3.0). See the EDRDG General Dictionary Licence Statement for details. Only JLPT relevant and common words have been included.
The application to obtain the data from jisho was made by John Charlton (coolmule0). It can be found here. The application is licensed under the GNU General Public License (V3.0). See the JLPT-N5-N1-Japanese-Vocabulary-Anki LICENSE page for details.

Stroke order The stroke order svg data for all kanji was obtained from KanjiVG. KanjiVG is copyright © 2009-2022 Ulrich Apel. The files are released under the Creative Commons Attribution-ShareAlike License (V3.0). See the Creative Commons license page for details. Only svg data for jouyou and JLPT relevant kanji has been included. Some version of the svg data has been adjusted by me for color blindness. See below for details.

Stroke color In order to make the different strokes as easy as possible to see for people with color blindness, I used this article by David Nichols as reference. The color pattern I chose is based on this article by Bang Wong.
